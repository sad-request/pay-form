# Инструкция по запуску

### Через Docker

- Запустить Docker

- В корневой директории выполнить

```
docker compose build
```

```
docker compose up
```

### Обычный запуск

- В директории /backend выполнить

```
npm i
```

```
npm run dev
```

- В директории /frontend выполнить

```
npm i
```

```
npm run dev
```

Разворачивается по адресу http://localhost:5173/
